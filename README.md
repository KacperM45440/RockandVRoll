# Inne mechaniki z gry
### 1. Punkt złapania przedmiotu określany jest dynamicznie
### 2. Fizyczne przedmioty można układać w stosy i je ze sobą nosić
![Dynamic grab](https://github.com/KacperM45440/RockandVRoll/assets/104095161/33decf27-8482-4ef2-b79b-7c8061fb6b30)
### 3. Menu jako hub łączący pozostałe poziomy, każdy poziom jako osobna scena, sceny zapisane jako prefaby
### 4. Dynamiczny podgląd następnego poziomu z pozycji drzwi w menu poprzez reflection probe'y
### 5. Przenoszenie postępu pomiędzy poziomami (zapisywanie, czy przedmiot został znaleziony)
![Prefabs](https://github.com/KacperM45440/RockandVRoll/assets/104095161/90eb55ec-f609-4625-9151-7a3f721929c6)
![Reflections](https://github.com/KacperM45440/RockandVRoll/assets/104095161/54163ec6-2cdb-468c-8666-83d23adbe1df)
### 6. Sockety - wkładanie przedmiotów na odpowiednie miejsca, zakładanie czapek na głowę, zjadanie jedzenia
